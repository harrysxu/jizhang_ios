## ⚠️ 重要提示

1. **URL 必须可公开访问**
   - App Store 审核时会检查 URL 是否可以访问
   - 确保 GitHub Pages 已启用并部署成功

2. **使用 HTTPS**
   - GitHub Pages 默认支持 HTTPS ✅
   - App Store 要求隐私政策必须使用 HTTPS

3. **保持更新**

## 📞 如果遇到问题

### GitHub Pages 无法访问

1. 检查 GitHub Pages 是否已启用
2. 确认分支和文件夹配置正确
3. 等待 1-5 分钟让部署生效
4. 清除浏览器缓存后重试

### App Store Connect 提示 URL 无效

1. 确认 URL 使用 HTTPS（GitHub Pages 默认支持）
2. 确认 URL 可以在浏览器中直接访问
3. 确认页面内容完整显示
4. 如果问题持续，可以尝试使用其他托管平台（Vercel、Netlify 等）

### 需要修改隐私政策内容

1. 编辑 `pages/privacy-policy.html` 文件
2. 更新"最后更新"日期
3. 提交并推送到 GitHub
4. GitHub Actions 会自动重新部署（1-2 分钟）
