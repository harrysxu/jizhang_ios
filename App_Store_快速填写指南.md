## 🚀 提交流程

### 1. 准备构建版本
```bash
# 在 Xcode 中
1. Product → Clean Build Folder (⌘+Shift+K)
2. 选择 "Any iOS Device"
3. Product → Archive
4. Distribute → App Store Connect
5. 等待处理（10-30分钟）
```

### 2. 在 App Store Connect 提交
1. 登录 https://appstoreconnect.apple.com
2. 选择 App
3. 点击"+"添加新版本
4. 填写所有信息（使用本文档）
5. 选择构建版本
6. 保存
7. **提交审核**

### 3. 等待审核
- 通常 1-3 个工作日
- 可在 App Store Connect 查看状态
- 审核通过后选择自动或手动发布
